Remove Vendor

Basic Path

  1. The Inventory Manager selects "View Vendors".

  2. The Inventory Manager selects one of the vendors.

  3. The Inventory Manager selects "Remove Vendors".

  4. The software checks that the vendor is not being used by any invoice or PO.

  5. The software removes the vendor.

  6. The software updates any listings of vendors.
