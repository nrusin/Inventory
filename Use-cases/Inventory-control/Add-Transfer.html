Add Transfer

Basic Path

  1. The use case begins when the Inventory Manager selects "Add-Transfer".

  2. The software checks whether one or two stockrooms are being used.

  3. Two stockrooms are being used.

  4. The software retrieves the units types being used.

  5. The software prompts for the direction of the transfer. By default the transfer is from the secondary stockroom to the primary stockroom.

  6. The Inventory Manager enters the direction of the transfer.

  7. For each product that the Inventory Manager adds to the transfer

    1. The software prompts for the Inventory Manager to identify the product, either by SKU, upc, or description.

    2. The Inventory Manager identifies the product.

    3. The software checks that the product was not priorly entered into the transfer.

    4. The software retrieves the default unit type for the product.

    5. The software prompts the Inventory for the number of items transfered and their unit type.

    6. The Inventory Manager enters the number of items and its unit type.

  8. The software checks that at least one product has been added to the transfer.

  9. The software adds the transfer. The transfer is unlinked to any particular pick order

  10. The software updates any listing of transfers.
