Update-Products

Basic Path

  1. The use case begins when the Inventory Manager selects "Update-Products".

  2. The software promts the Inventory Manager for the xlsx spreadsheet with the product information.

  3. The Inventory Manager enters the spreadsheet filename.

  4. The software checks that the spreadsheet is in the right format.

  5. The software reads the spreadsheet.

  6. For each product in the spreadsheet

    1. The software reads the product's description, upc, eoq and price.

    2. The software looks up the product using the upc.

    3. The software updates the product's description, upc, eoq, and price.
