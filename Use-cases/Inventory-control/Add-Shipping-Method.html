Add-Shipping-Method

Basic Path

  1. The use case begins when the Inventory Manager selects "Add-Shipping-Method"

  2. The software retrieves the previously added shipping methods.

  3. The software prompts for the name of the new shipping method.

  4. The Inventory Manager enters the new shipping method.

  5. The software adds the new shipping method.

  6. The software updates any listing of shipping methods.
