Add Department

Basic Path

  1. Include View Departments"

  2. The use case begins when the Inventory Manager selects "Add Department".

  3. The software retrieves the types of units that are being used.

  4. The software retrieves the department names and their corresponding department numbers that are being used.

  5. The software finds a default department number by adding one to the maxiumum current department number.

  6. The software prompts for the department's name, department number and department group.

  7. Using the types of units retrieved previously, the software prompts for the department's default unit type.

  8. The software checks that the department name and number are unique.

  9. The software creates a department using the department name, number, and default unit type.

  10. The software updates any listing of departments.
