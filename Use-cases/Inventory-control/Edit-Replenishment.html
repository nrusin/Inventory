Edit-Replenishment

Basic Path

  1. The Inventory Manager selects "View Replenishments".

  2. The Inventory Manager selects one of the replenishments to edit

  3. The software retrieves the replenishment information.

  4. The software displays the replenishment's beginning date.

  5. The software displays the replenishment's ending date.

  6. For each department within the replenishment

    1. The software retrieves the products of that department that are within the replenishment.

    2. The software shows the product's description, upc, price, and quantity sold.
