Edit Vendor

Basic Path

  1. The Inventory Manager selects "View Vendors".

  2. The Inventory Manager selects one of the vendor's.

  3. The software retrieves information about of vendor.

  4. The software prompts for the vendor's name.

  5. The software prompts for the vendor's id.

  6. The software prompts for alternate vendor's id's that replenishments use.

  7. The software prompts for the vendor's address. Fields of the the address are city, state, street, and zip code.

  8. The software prompts for the vendor's phone number.

  9. The software promtps for the vendor's UPC vendor number.

  10. The software prompts for the vendor's lead time.
