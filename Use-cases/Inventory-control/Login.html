Login

Basic Path

  1. The software prompts for the Inventory Manager's user name.

  2. The software prompts for the Inventory Manager's password. As the Inventory Manager enters the password it is shown using astricks.

  3. The Inventory Manager enters his or her user name and password.

  4. The software lookups the database's host name.

  5. The software lookups the database's dbname.

  6. The software checks that the database is a mysql database.

  7. The software attempts to connect to the database using the entered username and password.
