View Invoices

Basic Path

  1. The use case begins when the Inventory Manager selects "View Invoices".

  2. The software retrieves last ten invoices ordered by dates the invoice's were received.

  3. While the Inventory Manager has not selected an invoice

    1. The softare shows the invoices, the date they were received, and the PO to the Inventory Manager.

    2. At this point the Inventory may select a beginning date.

    3. The software prompts the Inventory Manager whether he or she wants to move the window of ten invoices back or forward.

    4. Depending on the current position of the invoice window, the software retrieves ten invoices backward or forward.

  4. The Inventory Manager selects an invoice.

  5. The software retrieves the invoice's vendor.

  6. The software retrieves the invoice's invoice number.

  7. The software retreives the invoice's PO.

  8. The software retrieves the invoice's shipment method.

  9. The software retrieves the date the invoice was received.

  10. The software retrieves the date the invoice was shipped.

  11. For each product shipped on the invoice

    1. The software retrieves the product's SKUs, description's, UPC, and cost

    2. The software retrieves the number of the products shipped and the unit type to show them.

  12. The software retrieves the product's ordered on the invoice.

  13. The software allows the Inventory Manager to view the PO.

  14. The software allows the Inventory Manager to select one of the products to show more detailed information.
