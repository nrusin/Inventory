Startup

Basic Path: MySQL

  1. The use case begins when the Inventory Manager starts the application.

  2. The software checks that a default database type has been set.

  3. The software prompts for whether to use a MySQL database or a Sqlite database.

  4. The Inventory Manager selects "MySQL".

  5. The software prompts for the hostname, dbname, user name and password.

  6. The software connects to the database.

  7. The software checks whether the database schema has been built. That is, for each table, the software checks that the table expected exists and has the right columns.

  8. The software builds the database schema.
